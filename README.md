# songulbabacan.com (Next.js 14)

Modern, SEO odakli ve performansli bir Next.js 14 (App Router) projesi.

## Kurulum

```bash
pnpm install
```

## Gelistirme

```bash
pnpm dev
```

Ardindan `http://localhost:3000` adresini acin.

## Uretim build

```bash
pnpm build
pnpm start
```

## Icerik yonetimi

Blog yazilari `content/blog` altindaki MDX dosyalarindan gelir. Yeni yazilar icin ayni
formatta yeni bir `.mdx` dosyasi ekleyin.

## Ortam degiskenleri

`.env.example` dosyasini referans alin.
